trait BaseDocument[T] extends Traversable[(String, BsonValue)] with Bson

Base Document trait.

A strictly typed Traversable[(String, BsonValue)] and provides the underlying immutable document behaviour. See immutable.Document or mutable.Document for the concrete implementations.

T

The concrete Document implementation
